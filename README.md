Phan Ansible role
=========
[![Build Status](https://travis-ci.com/devsoul/ansible-role-phan.svg?branch=master)](https://travis-ci.com/devsoul/ansible-role-phan)

If you are a PHP developer and you use Phan.

Requirements
------------
  - `php > 7.0` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.phan

Check if it's installed:
------
    phan --version
